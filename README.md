# Al Jazeera Ops Engine (POC)

A Proof of Concept for a C-level Central Operations Centre for a News & Media organization.

## Scope (POC)
- Executive Control Tower (Landing)
- Media Operations
- HR & Workforce
- Finance & Spend
- (Optional) Data Health

## Tech Stack
- BigQuery (POC datasets + tables)
- Looker (LookML + dashboards)
- Synthetic data generation (initial phase)

## Repository Structure
- /bigquery        : dataset creation + table DDL
- /looker          : LookML model, views, dashboard specs
- /synthetic_data  : synthetic generators + configs
- /docs            : architecture and design notes
